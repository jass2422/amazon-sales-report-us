## 📊 Project Overview

This project is an end-to-end **data analytics dashboard** built using **Microsoft Power BI**, focused on analyzing Amazon sales performance across the **United States**. It provides actionable insights into revenue trends, product performance, regional sales distribution, and customer behavior.

---

## 🎯 Objectives

- Analyze overall sales performance across different time periods
- Identify top-performing product categories and SKUs
- Visualize regional/state-wise sales distribution across the US
- Track key KPIs such as revenue, units sold, profit margins, and returns
- Enable data-driven decision-making for business stakeholders

## 📌 Key Features & Dashboards

### 1. 📈 Sales Overview
- Total Revenue, Units Sold, and Average Order Value (KPI Cards)
- Monthly/Quarterly sales trends (Line & Bar Charts)
- Year-over-Year (YoY) growth comparison

### 2. 🗺️ Regional Analysis
- State-wise sales distribution (US Map Visual)
- Top 10 performing states by revenue
- Regional breakdown of product categories

### 3. 🛒 Product Performance
- Best-selling products and categories
- Revenue contribution by category (Donut/Pie Chart)
- Product-level profitability analysis

### 4. 👤 Customer Insights
- Customer segmentation analysis
- Repeat vs. new customer trends
- Average spend per customer

### 5. 📦 Order & Returns Analysis
- Order fulfillment trends
- Return rate by category and region
- Delayed shipment analysis

## 🚀 How to Run This Project

### Prerequisites
- [Microsoft Power BI Desktop](https://powerbi.microsoft.com/en-us/downloads/) (Free) — Download and install

### Steps

1. **Clone this repository**
   ```bash
   git clone https://github.com/your-username/amazon-sales-report-us.git
   cd amazon-sales-report-us
   ```

2. **Open the Power BI file**
   - Launch **Power BI Desktop**
   - Click `File → Open` and select `Amazon_salse_report_for_united_states.pbix`

3. **Refresh Data (if needed)**
   - Go to `Home → Refresh` to reload the latest data
   - Update the data source path if prompted: `Home → Transform Data → Data Source Settings`

4. **Explore the Dashboard**
   - Navigate through different report pages using the tabs at the bottom
   - Use slicers/filters (date, region, category) to interact with the visuals
